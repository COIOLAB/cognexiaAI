## 🔧 Deployment Commands

### **Installation**
```bash
# Clone repository
git clone <repository-url>
cd modules/06-procurement

# Install dependencies
npm install

# Build application
npm run build

# Run database migrations
npm run migration:run

# Seed initial data (optional)
npm run seed
```

### **Production Start**
```bash
# Start application
```
