## 🚀 **Getting Started**

### **🔧 Prerequisites:**
- Node.js 18+
- NestJS Framework
- PostgreSQL Database
- Redis Cache
- WebSocket Support

### **⚙️ Installation:**
```bash
# Install dependencies
npm install

# Configure environment
cp .env.example .env

# Initialize database
npm run db:migrate

# Start the application
npm run start:dev
```

### **📡 WebSocket Connection:**
```typescript
// Connect to HR WebSocket
const socket = io('ws://localhost:3000/hr-socket');

// Subscribe to employee updates
socket.emit('subscribe-hr-updates', {
  departments: ['IT', 'Finance'],
  employees: ['EMP_001', 'EMP_002'],
  reviews: ['REV_001']
});
```
